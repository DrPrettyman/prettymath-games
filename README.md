# PrettyMath Games

Educational mathematics games for children.

## Installation

```bash
npm install prettymath-games
```

## Usage

```jsx
import { AngleEstimationGame, FractionEstimationGame } from 'prettymath-games';

<AngleEstimationGame />
<FractionEstimationGame />
```
